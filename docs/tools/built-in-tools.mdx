# Built-in Tools

All 40 built-in tools organized by category. Filter by trait, click any card to expand.

<ToolCatalog />

## Tool Availability by Agent Type

Not all agents have access to all tools. Sub-agents get filtered tool sets:

| Agent Type | Available Tools |
|------------|----------------|
| `general-purpose` | All tools |
| `Explore` | Read-only tools (Read, Glob, Grep, WebFetch, WebSearch) — no Edit, Write, Bash |
| `Plan` | Read-only tools — no Edit, Write, Bash |
| `code-simplifier` | All tools |
